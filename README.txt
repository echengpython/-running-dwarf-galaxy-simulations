Welcome!

This is your own little computing environment -- there isn't much here at the moment, except the 
'getting started' scripts from the TNG data release:

  illustris_python/
  illustris_idl/
  illustris_julia/
  illustris_matlab/

There are a few Jupyter notebook-based examples (in Python) which may be useful to get started:

  examples/

And the actual simulation data, which you'll find in:

  sims.illustris/
  sims.TNG/
  sims.other/

Note that the simulation directories are read-only locations, so you'll want to make a new folder 
inside the home directory to organize and store your analysis. Please be careful to **not** copy 
the actual simulation files, you should read them from their existing locations.

Data in your home directory (/home/tnguser/) will persist across sessions.

You can install software as needed at the terminal (e.g. 'pip install astropy'), which will by 
default install into your home directory. For more sophisticated package management, it is 
suggested to set up a new mamba/conda environment (e.g. 'mamba' command).
